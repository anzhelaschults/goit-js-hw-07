# goit-js-hw-07

Homework assignment for GoIT JavaScript course. Topic 12: DOM Model. Events.
Домашнє завдання курсу JavaScript від GoIT. Тема 12: Модель DOM. Події.

## Tasks / Завдання

- **Task 1** — counts and logs the number of categories and items in each category from ul#categories / підраховує та виводить кількість категорій і елементів у кожній категорії зі списку ul#categories
- **Task 2** — creates an image gallery from an array of objects and adds it to ul.gallery in one DOM operation / створює галерею зображень з масиву об'єктів і додає її в ul.gallery за одну операцію
- **Task 3** — listens to input event on input#name-input and updates span#name-output with the current value or "Anonymous" if empty / слухає подію input та оновлює span#name-output поточним значенням або "Anonymous" якщо поле порожнє
- **Task 4** — handles login form submission, validates fields, logs credentials object and resets the form / обробляє відправку форми логіна, валідує поля, виводить об'єкт з даними і скидає форму
- **Task 5** — changes body background color on button click and displays the hex value in span.color / змінює колір фону body при кліку на кнопку і відображає hex значення у span.color
- **Task 6** — creates colored boxes of increasing size on Create button click, clears them on Destroy / створює кольорові квадрати зростаючого розміру при кліку Create і видаляє їх при кліку Destroy
